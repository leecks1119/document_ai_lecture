### Python 환경 설정
```bash
# Python 환경 설정
pip install pytesseract opencv-python pillow
pip install easyocr paddlepaddle paddleocr
pip install openai langchain transformers

# OCR 엔진 설치 (Ubuntu 기준)
sudo apt install tesseract-ocr tesseract-ocr-kor
```
