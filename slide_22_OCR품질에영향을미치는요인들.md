# 슬라이드 22: OCR 품질에 영향을 미치는 요인들

**리딩 메시지**: "OCR 실패 요인을 사전에 파악하여 예방 중심의 시스템을 구축하겠습니다"

## 환경적 요인

- **이미지 품질**: 해상도 (최소 300DPI 권장), 조명, 그림자
- **문서 상태**: 구겨짐, 찢어짐, 얼룩, 노후화
- **스캔 설정**: 색상 모드, 압축률, 파일 포맷

## 문서적 요인

- **폰트**: Sans-serif vs Serif, 폰트 크기 (최소 8pt)
- **레이아웃**: 다단 구성, 텍스트 밀도, 여백
- **언어**: 한글/영어 혼재, 특수문자, 수식

## 기술적 요인

- **OCR 엔진 선택**: 언어별 특화 모델
- **전처리 파이프라인**: 적절한 필터 조합
- **후처리 규칙**: 도메인 특화 검증 로직
